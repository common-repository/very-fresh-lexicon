== Description ==

“Very fresh Lexicon" is a simple yet customizable plugin to create a lexicon in wordpress. 
Have your own lexicon or glossary in minutes. Use Flexicon to create glossary like custom post-types the easy way. Define your custom post-type in Flexicon, for example "dictionary" or "glossary" and start creating items for it right away. Having a knowledgebase is beneficial for the user and your search engine results. This plugin can do it all and even has settings for custom CSS. There is no pro version or subscription model.

== Installation ==

AUTOMATIC INSTALLATION

1. Go to Plugins > Add New
2. Type “Very fresh Lexicon” in the search field and click “Search Plugins”
3. Click “Install Now” to install
4. Click “Activate”


MANUAL INSTALLATION

1. Download a copy of the “Very fresh Lexicon” plugin
2. Upload freshPodcaster to the /wp-content/plugins/ directory
3. Activate the plugin through the “Plugins” menu in WordPress
4. Use the fresh Podcast menu from the sidebar
